# My-Swing-Day-Trading-Bot

Termux-ready Python signal bot scanning XAUUSD, XAGUSD, BTCUSD, GBPUSD, USDJPY on M30/H1/H4.
Outputs signals via Telegram (ENTRY, SL, TP, reason).

Files:
- main.py — orchestrator
- requirements.txt — pip packages
- config.example.json — template config (edit locally)
- src/ — code modules
- src/strategies — strategy modules (examples + stubs)

Setup & run instructions are in README (Termux section).
